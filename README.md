# FLUX Baton
Generational context handoff for agents made easy.

[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](https://opensource.org/licenses/MIT)
> *An agent's work outlives its context window. The baton carries everything the next agent needs.*

* Enables seamless handoff between agent generations
* Implements quality gates for ensured consistency
* Employs a federated marching band model for coordination

Quick start: Clone the repo and run the example use case.

Architecture: Microservices-based, with modular components.

Part of [Cocapn Fleet](https://github.com/SuperInstance/cocapn-fleet)
